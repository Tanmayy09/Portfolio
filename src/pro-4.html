PROJECT OVERVIEW

E-commerece website is a web-based application designed to manage online inventory for the customers. The E-Commerce Website is a dynamic platform for buying products online. It provides users with a seamless shopping experience and allows businesses to manage their inventory, orders, and customer interactions efficiently. It is an Open-Source project which is a simple and clean multi-page website for users.

It has many features which include :-

  1. User Management
    • Secure login and registration for customers.
    • Role-based access for admin and users.
  2. Product Management
    • Display product categories, details, and images.
    • Search, filter, and sort products by price, category, or ratings.
  3. Shopping Cart
    • Add, update, and remove items.
    • Address input for delivery.
    • View total cost and estimated delivery.
  4. Misc
    • View past and current orders.
    • Integration with payment gateways for secure transactions.
    • Responsive design for seamless access across devices.
